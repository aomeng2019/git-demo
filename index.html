将本地仓库推送到远程仓库命令 1.git push 远程仓库地址 分支名称 2.git push 远程仓库地址别名 分支名称 3.git push -u 远程仓库地址别名 分支名称 -u 记住推送地址及分支 下次推送只需要直接输入git push即可 为远程仓库添加别名 4.git remote add 远程仓库地址别名 远程仓库地址 remote是远程的意思 远程仓库地址别名一般为origin 第一次推送到远程仓库时要登陆，登陆后的信息就保存在你自己电脑的windows系统中的凭据管理中。下一次要将本地仓库推送到远程仓库时，windows系统自动将登陆信息发送给github.com并登陆。则就不需要用户再一次登陆了 克隆远端数据仓库到本地 git clone 仓库地址 拉取远程仓库中最新的版本：git pull 远程仓库地址 分支名称 git pull只是拉取远程仓库中最新的版本，这个最新的版本是和本地仓库中的最新版本进行比较的。 所以git pull是在已经有本地仓库的情况下进行的。 git clone是完全克隆远程仓库，是在没有本地仓库的基础上进行的 我是index.html文件
